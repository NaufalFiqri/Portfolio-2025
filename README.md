## About This Project

This portfolio was designed to **highlight my personality and life journey** while demonstrating my capabilities as a frontend developer. It combines **modern design aesthetics** with interactive elements, animations, and a smooth user experience.  

It is **fully responsive** and optimized for desktop and mobile browsing.

---

## Features

- **Homepage:** A personal homepage for the website.  
- **Skills Constellation:** Highlights the technologies and tools I’ve mastered up to 2025.  
- **Crystal Vault:** Displays selected projects with screenshots, live demos, and GitHub links.  
- **Life Loadout:** Showcases who I am, the experiences I’ve gained, and the skills I’ve built so far.
